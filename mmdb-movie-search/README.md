# MongoDB Movie Database (MMDB)

A modern web application for searching movies from MongoDB Atlas using the sample_mflix database.

## Features

- **Full Text Search** - Search across multiple fields (implement with MongoDB Atlas Search)
- **Exact Match Search** - Search with field selection (Title, Cast, Plot)
- **Sorting** - Sort results by Year or IMDb Rating
- **Dual View** - Toggle between List and Tile view
- **Pagination** - Browse results 10 per page
- **Movie Details Modal** - View detailed information about each movie
- **Dark Green & White Theme** - Professional color scheme

## Prerequisites

Before you begin, ensure you have:

- **Node.js** (v14 or higher) - [Download here](https://nodejs.org/)
- **MongoDB Atlas Account** - [Create free account](https://www.mongodb.com/cloud/atlas)
- **sample_mflix Database** - Available as a free sample in MongoDB Atlas
